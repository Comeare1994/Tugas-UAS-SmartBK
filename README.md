# 
#📱 SmartBK

SmartBK adalah aplikasi Android yang dirancang untuk membantu guru Bimbingan dan Konseling (BK) dalam mengelola data siswa dan layanan BK secara lebih efektif.

## 📌 Fitur Aplikasi

- Login
- Dashboard
- Data Siswa
- Konseling
- Home Visit
- Surat
- Pelanggaran
- Absensi
- Rekap Data
- Pencarian Global
- Pengaturan

## 🛠 Tech Stack

- Java
- Android Studio
- SQLite Database
- Material Design Components
- Gradle

## 📱 Cara Menjalankan Aplikasi

### Persyaratan

- Android Studio
- Minimum SDK 24 (Android 7.0)
- Target SDK 36

### Langkah Instalasi

1. Clone repository ini.
2. Buka project menggunakan Android Studio.
3. Tunggu proses Gradle Sync selesai.
4. Jalankan aplikasi menggunakan Emulator atau perangkat Android.
